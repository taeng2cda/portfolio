홈페이지 소개

OTT 서비스 요금 공유를 목적으로 파티를 찾고자하는 유저들을 위한 커뮤니티 사이트

  • OTT 파티를 찾고자하는 유저들을 위해 기본적인 커뮤니티 기능을 지원합니다
  • 자동파티매칭 기능을 지원하여 유저들간의 파티 매칭과 관리를 용이하게 해줍니다

메인페이지

OTT조 프로젝트의 메인페이지 입니다
메인페이지는 아래의 기능이 있습니다

  • 시큐리티를 사용하여 회원로그인,관리자 로그인이 가능합니다
  • 채팅기능이 구현되어 있습니다
  • 알림기능이 추가되며 있으며 자기 글,댓글에 좋아요를 누를 경우 카운트가 올라갑니다
  • 게시글을 작성,수정,글쓰기,목록보기가 가능합니다

관리자/메인페이지

이제부턴 제가 담당한 관리자 페이지를 소개합니다
해당 페이지는 아래의 기능이 있습니다

  • 파티매칭 상태를 4가지 형태로 나눴으며 직관적으로 확인이 가능합니다
  • 매출 현황을 차트로 표현
  • 전체 디자인은 부트스트랩 템플릿을 사용했습니다
  • 페이지를 조각내어 타일즈 API를 사용해 다른페이지 이동시 헤더푸터가 같이 이동됩니다

관리자/방문자 조회

홈페이지에 접속하면 이벤트 리스너가 실행되어 접속한 사람의 IP를 가져와 데이터를 DB에 추가하는 기능을 구현했습니다 그렇게 수집한 데이터로 차트를 그려봤습니다
해당 페이지는 아래의 기능이 있습니다

  • Datepicker를 활용 하여 해당 날의 정보를 불러옴
  • ajax를 통해 차트.js 라이브러리를 활용함
  • dataTables라이브 러리를 통해 페이징 처리를 구현했습니다

관리자/통계페이지

해당 페이지는 회원 데이터를 활용 하여
차트.js를 통해 아래의 3가지 통계를 구현했습니다

  • 7일간 회원 가입수 (라인 차트)
  • 회원 나이 분류 (도넛 차트)
  • 탈퇴 회원 및 블랙유저 카운터 통계(막대 차트)

관리자/결제 조회

결제 진행된 데이터를 활용하여 차트를 그려냈습니다
차트는 결제완료,결제취소,결제횟수 주간단위로 확인이 가능하며 상세데이터를 페이징처리를 하여 만들었습니다.

  • Chart.js를 활요해 차트를 그려냈습니다
  • dataTables라이브 러리를 통해 페이징 처리를 구현했습니다

관리자/sms문자발송

해당 페이지는 실제 문자 발송이 가능한 페이지입니다.
카페24 SMS호스팅을 연동하여 실제 건수마다 문자 발송이 되며 MMS,LMS 종류마다 차감 건수가 다릅니다

  • 카페24 SMS호스팅을 연동하여 실제 문자 발신이 가능
  • 예약메세지 기능도 존재합니다.
